LEAD GENERATION

3. Naya lead kaise banayein

  1. Sidebar mein Sales → Lead Generation open karein.
  2. Company step mein communication mode, status, company, industry, EPR category, Applicant Type, subcategory aur services fill karein.
  3. Address step mein complete registered/correspondence address add karein.
  4. Contact step mein contact person, mobile aur email verify karke enter karein.
  5. Assign step mein responsible team/user select karein.
  6. Save Draft se incomplete record bachayein; complete data par Next Step/Save karein.
Good data practice: Company name official documents ke same rakhein; mobile/email typing error check karein; PIN code aur state match karein.

Applicant Type dependent selection

Applicant TypeAvailable subcategories
PIBOProducer, Brand Owner, Importer
SIMPProducer (Small & Micro), Importer of Raw Material, Manufacturer of Raw Material, Seller
PWPRecycler, Waste to Energy, Waste to Oil, Cement Co-processing
Pehle Applicant Type choose karein. Uske baad usi type ki child category dikhegi. Parent change karne par old child selection clear ho jayegi.

Add New Category

Child dropdown mein Add New Category choose karein. Category current Applicant Type ke andar save hogi. Same parent mein case-insensitive duplicate name allowed nahi hai.

EXCEL IMPORT

4. Bulk lead upload

  1. Lead Generation page par Upload Excel click karein.
  2. Supported Excel file select karein aur headers check karein.
  3. File preview/row count confirm karein.
  4. Import Drafts click karke processing complete hone dein.
  5. Success aur failed row summary padhein; failed rows correct karke sirf corrected data re-upload karein.

Recommended Excel columns

SectionColumns
CompanyCompany, Status, EPR Category, Applicant Type, PIBO Subcategory, Services Offered, Industry Type
AddressAddress, City, PIN, State
ContactContact Person, Mobile, Email
Legacy support: Purani file mein sirf “PIBO Category” ho to known value se system parent infer karne ki koshish karta hai. New files mein Applicant Type aur PIBO Subcategory alag columns use karna best hai.
Import se pehle: merged cells, decorative title rows, blank mandatory values aur repeated header rows remove karein. Category spelling exact rakhein.

Agar bulk upload fail ho

  • Required columns aur column names check karein.
  • Applicant Type PIBO/SIMP/PWP mein se ho.
  • Subcategory selected parent ki hi ho.
  • Mobile/email format aur excessively long text correct karein.
  • Error message mein diye first failed row ko Excel mein locate karein.

QUOTATIONS

5. Quotation create aur manage karein

  1. Sales → Add Quotation open karein.
  2. Existing lead/client choose karein ya required customer details fill karein.
  3. Applicant Type aur matching subcategory select karein.
  4. Service/category, financial year, quantity/unit, basic amount aur applicable commercial details add karein.
  5. Terms, validity, contact details aur all totals review karein.
  6. Save/submit karein. Required approval flow ho to record Pending Approval mein jayega.
  7. Authorized approval ke baad final quotation download/share karein.

PO kya hai?

PO = Purchase Order. Buyer dwara seller ko issue kiya official order document, jisme service/product, price, terms aur delivery details hoti hain.

Revision rule

Price/terms badalne par approved document overwrite na karein. Proper revised quotation create karke fresh approval follow karein.

Final check before submission

  • Company legal name and address correct
  • Applicant Type + subcategory match
  • Services, FY, unit and amount correct
  • Tax/total and payment terms checked
  • Quotation date and validity checked
  • Contact person and mobile/email verified
Important: Downloaded quotation ko share karne se pehle PDF visually check karein—logo, quotation number, table, totals, terms aur footer clear hone chahiye.

DAILY PRODUCTIVITY

7. Dashboard, notification aur calendar

Dashboard ko sahi tarah use karein

  • Dashboard open karke required Operations ya Sales view choose karein.
  • Sales widgets mein time filter ko Month par set karke current month select karein.
  • Widget number par depend karne ke saath “View Full Report” se detailed records bhi verify karein.
  • Zero/No data found ka matlab selected filter mein matching data nahi bhi ho sakta hai.

Notifications

  • Bell icon ka unread count regularly clear karein.
  • Approval, assignment aur record update notification open karke linked item par action lein.
  • Notification read karna action complete karne ke barabar nahi hai.

TROUBLESHOOTING

8. Common problems aur solutions

ProblemKya karein
Page slow/loadingCurrent action ko repeat-click na karein. Internet check, 15–30 sec wait, then one refresh. Exact page/time note karein.
Save nahi ho rahaRed validation text aur required * fields check karein. Dropdown ki valid option choose karein.
Wrong child categoriesApplicant Type reselect karein. Parent badalne par child dobara choose karna required hai.
Excel rows failedFirst failed row message padhein; headers, mandatory data aur category relation correct karein.
Duplicate categorySame Applicant Type ke under existing spelling search/select karein; capitalization badal kar duplicate create na karein.
OTP nahi aayaEmail/spam check, latest OTP use, admin se active account verify.
Permission/access deniedApna assigned role/team admin se verify karayein.
PDF wrong layoutFresh PDF generate karein; browser preview ke bajay downloaded file check karein, then screenshot + quotation no. share karein.
Support ko kya bhejein: Screenshot, page name, record/quotation ID, exact error message, date-time, aur issue reproduce karne ke steps. Password, OTP ya sensitive credentials kabhi share na karein.

Safe retry method

  1. Error message ka screenshot lein.
  2. Check karein record already save to nahi hua.
  3. Required correction karein.
  4. Save/Import button sirf ek baar click karein.

9. One-page daily checklist

Start of day

  • Notifications read
  • Pending approvals reviewed
  • Today’s calendar checked
  • Assigned leads prioritized

Before saving a lead

  • Company spelling correct
  • Applicant Type + child valid
  • Contact verified
  • Address and assignment complete

Before quotation

  • Correct customer selected
  • Service and FY correct
  • Amounts/totals checked
  • Terms and validity reviewed

End of day

  • Follow-ups updated
  • Completed tasks closed
  • Drafts reviewed
  • Tomorrow’s calendar prepared

Remember

CRM tab useful hota hai jab data complete, correct aur timely ho. Har call, follow-up aur status ko same day update karein.

Short workflow: Lead banao → data verify karo → quotation prepare karo → approval complete karo → Client Master update karo → calendar follow-up rakho.
